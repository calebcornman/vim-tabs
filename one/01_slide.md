# Vim Tabs #

!SLIDE bullets
# Opening tabs from the command line#

* vim -p file1 file2 etc

!SLIDE bullets incremental

# Opening inside vim

* :tabnew file
* Ctrl-w gf
* Ctrl-w T

!SLIDE bullets incremental

# Closing tabs

* :tabc
* :q

!SLIDE smbullets incremental

# Navigate between tabs

* gt (can modify with tab number)
* gT
* :tabp
* :tabn
* :tabfirst
* :tablast

!SLIDE bullets

# Moving tabs

* :tabm position
